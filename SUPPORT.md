# Support

## Where to get help

For this project, GitHub Issues is the main support channel.

- Bug report: use the bug report template
- Feature idea: use the feature request template
- Release download: see the [Releases page](https://github.com/louisYKL/TV-BOX-DolbyVision-HDR10-Edition/releases)

## What to include

If you want a useful answer quickly, include as much of this as possible:

- app version
- build variant: `java32`, `java64`, or `hisense32`
- device brand and model
- Android version
- stream container: MP4 / MKV / TS / WebM
- codec details if known
- SDR / HDR10 / HDR10+ / Dolby Vision status if known
- whether subtitles, fullscreen, source switching, or passthrough are involved
- reproduction steps

## Playback issues

For playback issues, the most helpful reports are the ones that separate the problem clearly:

- loads but has no picture
- has picture but no sound
- HDR does not activate
- source switch crashes
- fullscreen behavior is wrong
- subtitles exist but do not show
- only a specific container or codec fails

## Security or private data

Do not post:

- private source subscriptions
- tokens or cookies
- personal streaming URLs
- account credentials

If the report is security-sensitive, read [SECURITY.md](SECURITY.md) first.

## Scope

This repository is focused on the app itself:

- playback routing
- TV interaction
- subtitle and audio behavior
- HDR / Dolby Vision capability handling
- build variants and release packaging

It does not provide bundled media catalogs, live sources, or subscription content.
